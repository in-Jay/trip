# 여행사 홈페이
여행 정보를 등록하고 예약할 수 있는 여행사 페이지
<br><br>
## 프로젝트 목표
- 네이버 API를 사용하여 여행과 관련된 정보를 제공 받아 홈페이지를 제작하였습니다.
- Spring boot를 사용하여 홈페이지를 디자인하고 백엔드 기능을 구현하여 서비스를 제공하려 했습니다.
- 회원가입, 로그인, 여행지 등록, 예약 등의 기능을 구현했습니다.
<br><br>
## 제작기간
- 2024.11.18 ~ 2025.11.29
<br><br>
## 사용기술
- Spring boot
- Java
- JavaScript
- Thymleaf
- MariaDB
- HTML
- CSS
